# Bo so do sequence PlantUML

Thu muc nay chua cac so do sequence cho cac luong nghiep vu chinh cua FreshFood AI.

## Danh sach file

- `01_sequence_dat_hang_va_thanh_toan.puml`: Luong mua hang, ap dung voucher, dung diem, tao don va tru kho
- `02_sequence_xac_minh_do_tuoi_sau_giao_hang.puml`: Luong hau kiem sau giao hang, AI phan tich anh tai frontend, gui xac minh va cong thuong

## Ghi chu

- Cac sequence duoc tong hop tu source code thuc te trong `frontend/src/pages`, `frontend/src/services`, `backend/api/orders.py`, `backend/api/rewards.py`, `backend/utils/freshness.py`, `backend/utils/inventory.py`.
- Ten duoc viet khong dau de tranh loi encoding khi render PlantUML.
